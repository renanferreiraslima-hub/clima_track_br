# 🌦️ Weather App

Aplicação web de previsão do tempo que permite buscar cidades e visualizar informações climáticas em tempo real.

## 🚀 Funcionalidades

- 🔍 Busca de cidades com sugestões automáticas
- 🌡️ Exibição de temperatura atual
- ☁️ Condições climáticas (nublado, ensolarado, chuva, etc.)
- 💧 Umidade do ar
- 🌬️ Velocidade do vento
- 📍 Localização por nome da cidade
- ⚡ Atualização em tempo real via API

## 🛠️ Tecnologias utilizadas

- React.js
- JavaScript (ES6+)
- CSS3
- API de clima (ex: OpenWeatherMap)
